# 3TH

3TH ist ein System zur Darstellung von GHOST‑SCAN‑Lagen und Zeit‑Raster (3×11).

## Dateien

- index.html – GHOST‑SCAN
- time_3x11_scroll_zoom.html – Zeit‑Raster
- fin.e.js – Kernel
- RESPO/ – Cluster‑Daten
- ID.html – ID‑Matrix

## Funktion

3TH lädt RESPO‑Cluster und zeigt:

- ID
- LAGE
- STATUS
- RESULT
- OUT
- WAHRHEIT
- EDIT
